# 🎬 Watchlist

Watchlist is a Python desktop application for managing, tracking, and reviewing movies. It provides a modern, intuitive graphical interface to help you build your own personal movie library, track what you’ve watched, write reviews, and even get smart recommendations.

To run Watchlist properly, you’ll need to configure a `.env` file in the root directory of the project.

Create a file named `.env` and add the following:

```env

email_address=your_email@example.com
